# Orchestration

1. Collect a sanitized JSON fixture.
2. Run `npm run smoke` or the CLI directly for JSON evidence.
3. Run the CLI with `--format=markdown` when preparing a walkthrough, post, or launch note.
4. Review generated timing, artifact plan, approval gate, limitations, and next steps.
5. Store output with the agent run evidence.
6. Only perform external actions in a separate approved workflow.
